# procrastiSlides
***Web-based Markdown Presentation Generator utilizing `pandoc` and `pdflatex`***

&nbsp;

## `LaTeX` Dependencies for Basic Functionality
- texlive-bin
- texlive-core
- texlive-fontsextra
- texlive-formatsextra
- texlive-langchinese
- texlive-langextra
- texlive-langjapanese
- texlive-latexextra
- texlive-pictures
